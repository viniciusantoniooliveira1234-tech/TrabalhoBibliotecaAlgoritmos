"# TrabalhoBibliotecaAlgoritmos" 
#O trabalho final do CCR Algoritmos e Programação consistirá no desenvolvimento de um sistema de controle de biblioteca. O sistema deverá permitir o cadastro e gerenciamento de livros.
**Cada livro deve conter:**
*Titulo
*Autor
*Ano de publicação
*Código (ID único)
*Status (disponível ou emprestado)
**Funcionalidades (Menu)
**O sistema deverá apresentar um menu com as seguintes opções:
*Cadastrar livro
*Consultar livro
*Alterar dados
*Remover livro
*Listar todos
*Realizar empréstimo
*Realizar devolução
*Sair
