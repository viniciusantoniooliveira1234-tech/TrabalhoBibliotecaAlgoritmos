# TrabalhoBibliotecaAlgoritmos

## Sobre o Projeto

O trabalho final do CCR **Algoritmos e Programação** consistiu no desenvolvimento de um sistema de controle de biblioteca.

O sistema permite o cadastro e gerenciamento de livros, simulando funcionalidades básicas de uma biblioteca real.

---

## Dados de Cada Livro

Cada livro possui as seguintes informações:

* **Título**
* **Autor**
* **Ano de publicação**
* **Código** *(ID único)*
* **Status** *(Disponível ou Emprestado)*

---

## Funcionalidades do Sistema

O sistema apresenta um menu com as seguintes opções:

*  Cadastrar livro
*  Consultar livro
*  Alterar dados
*  Remover livro
*  Listar todos os livros
*  Realizar empréstimo
*  Realizar devolução
*  Sair

---

## Tecnologias Utilizadas

O sistema foi desenvolvido utilizando:

* **Python**
* **Tkinter** *(biblioteca de interface gráfica do Python)*

---

##  Observações

Não utilizei IA para gerar código do projeto.

Utilizei ferramentas de IA apenas como apoio para:

* Encontrar erros específicos
* Aprender conceitos da biblioteca **Tkinter**
* Tirar dúvidas durante o desenvolvimento

Todo o desenvolvimento e lógica do sistema foram feitos manualmente.
